DIF_WORDWRAP

главная | фаги элементов диалога | Dialog API
Флаг DIF_WORDWRAP усекает очередное слово, если оно не вмещается по ширине элемента.

Элементы

Флаг DIF_WORDWRAP имеет смысл для следующих элементов диалога:
ЭлементОписание
DI_TEXT текстовая строка

Замечания

  1. При указании этого флага, плагин должен обязательно корректно заполнить координаты X1,X2,Y1,Y2.
  2. Совместно с флагом DIF_WORDWRAP можно применять только флаги DIF_RIGHTTEXT, DIF_CENTERTEXT и DIF_SHOWAMPERSAND.
