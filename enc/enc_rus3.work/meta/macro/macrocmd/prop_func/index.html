Существуют свойства и функции как общего назначения, так и предназначенные для отдельных макрообластей.

  • Общие
  • Зависящие от контекста исполнения
  • Для панелей
  • Для командной строки
  • Для редактора
  • Для внутренней программы просмотра
  • Для диалогов
  • Для меню и списков
  • Для системы помощи

Замечания

  1. Логические свойства используются в качестве условий в операторах $If и $While.
  2. Параметры функций, обозначенные в квадратных скобках, можно не указывать. Значение по умолчанию см. в описаниях функций.
