PluginsControl

главная | сервисные функции
Функция PluginsControl позволяет управлять плагинами. 
INT_PTR WINAPI PluginsControl(
  HANDLE hHandle,
  enum FAR_PLUGINS_CONTROL_COMMANDS Command,
  int Param1,
  void *Param2
);

Параметры

hHandle
Смотрите описание Command. (в настоящее время значение игнорируется).
Command
Может быть одним из следующих значений (перечисление FAR_PLUGINS_CONTROL_COMMANDS):
КомандаОписание
PCTL_LOADPLUGIN Загрузить плагин.
Param1 должен быть установлен в значение PLT_PATH (перечисление FAR_PLUGIN_LOAD_TYPE).
Param2 указывает на строку, содержащую полный путь dll-файл плагина.
Функция возвращает TRUE, если плагин удалось загрузить.
PCTL_FORCEDLOADPLUGIN Загрузить плагин сразу в память (для межплагинового взаимодействия).
Param1 должен быть установлен в значение PLT_PATH (перечисление FAR_PLUGIN_LOAD_TYPE).
Param2 указывает на строку, содержащую полный путь dll-файл плагина.
Функция возвращает TRUE, если плагин удалось загрузить.
PCTL_UNLOADPLUGIN Выгрузить плагин.
Param1 должен быть установлен в значение PLT_PATH (перечисление FAR_PLUGIN_LOAD_TYPE).
Param2 указывает на строку, содержащую полный путь dll-файл плагина.
Функция возвращает TRUE, если плагин удалось выгрузить.
Param1
Указывает на Параметр1 команды. Смотрите описание Command.
Param2
Указывает на Параметр2 команды. Смотрите описание Command.

Возвращаемое значение

Смотрите описание Command

Замечания

Перечисление FAR_PLUGIN_LOAD_TYPE:
ТипОписание
PLT_PATH Полное имя dll-файла плагина, включая папку.

Пример

const wchar_t *PlugPath = L"C:\\FAR2\\Plugins\\AutoWrap\\AutoWrap.dll";

//загрузим либо выгрузим плагин
if (bLoad || bUnload)
{
  if (bLoad)
    Info.PluginsControl(INVALID_HANDLE_VALUE,PCTL_LOADPLUGIN,PLT_PATH,PlugPath);
  else
    Info.PluginsControl(INVALID_HANDLE_VALUE,PCTL_UNLOADPLUGIN,PLT_PATH,PlugPath);
}
