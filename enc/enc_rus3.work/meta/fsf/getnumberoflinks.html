GetNumberOfLinks

главная | стандартные функции
Функция FSF.GetNumberOfLinks возвращает количество жёстких ссылок (hard links) для заданного файла. 
size_t WINAPI GetNumberOfLinks(
  const wchar_t *Name
);

Параметры

Name
Имя требуемого файла.

Возвращаемое значение

В случае успеха функция возвращает количество жёстких ссылок или 0. Для файловых систем, отличных от NTFS это значение всегда равно 1.
