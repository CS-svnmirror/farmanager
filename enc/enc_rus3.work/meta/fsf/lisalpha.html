LIsAlpha

главная | стандартные функции
Функция FSF.LIsAlpha проверяет, является ли символ с указанным кодом буквой. 
int WINAPI LIsAlpha(
  wchar_t Ch
);

Параметры

Ch
Код символа для проверки.

Возвращаемое значение

Если указанный символ является буквой, то вернётся TRUE, иначе - FALSE.
