KeyBarLabel

главная | структуры | Panel API
Структура KeyBarLabel содержит новые заголовки функциональных клавиш, передается в Far Manager через структуру KeyBarTitles. 
struct KeyBarLabel
{
  struct FarKey Key;
  const wchar_t *Text;
  const wchar_t *LongText;
};

Элементы

Key
Клавиша, структура типа FarKey.
Text
Новый заголовок функциональной клавиши. Для сохранения стандартного заголовка Far Manager установите значение в NULL.
LongText
Новый длинный заголовок функциональной клавиши. Для сохранения стандартного заголовка Far Manager установите значение в NULL.
