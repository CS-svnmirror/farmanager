ViewerSelect

главная | структуры | Viewer API
Структура ViewerSelect используется для выделения области в программе просмотра Far Manager. 
struct ViewerSelect
{
  __int64 BlockStartPos;
  int  BlockLen;
};

Элементы

BlockStartPos
Начало выделения - в символах, а не в байтах. Это означает, что если программа просмотра находится в режиме Unicode, то BlockStartPos будет равно (позиция в файле в байтах / 2).
BlockLen
Длина выделения в символах.
