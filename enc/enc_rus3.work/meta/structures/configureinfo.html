ConfigureInfo

главная | структуры
Структура ConfigureInfo используется Far Manager'ом в функции ConfigureW. 
struct ConfigureInfo
{
  size_t StructSize;
  const GUID *Guid;
};

Элементы

StructSize
Это поле содержит размер структуры ConfigureInfo.
Guid
GUID выбранного пункта, в списке пунктов, экспортированных плагином в меню "Параметры внешних модулей".
