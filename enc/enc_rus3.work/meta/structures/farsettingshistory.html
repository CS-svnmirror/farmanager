главная | структуры
Структура FarSettingsHistory используется в функции SettingsControl для управления настройками. 
struct FarSettingsHistory
{
  const wchar_t *Name;
  const wchar_t *Param;
  GUID PluginId;
  const wchar_t *File;
  FILETIME Time;
  BOOL Lock;
};
