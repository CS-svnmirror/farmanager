KeyBarTitles

главная | структуры | Panel API
Структура KeyBarTitles позволяет переопределять заголовки функциональных клавиш. Массив этих структур передаётся в Far Manager функцией GetOpenPanelInfoW. 
struct KeyBarTitles
{
  size_t CountLabels;
  struct KeyBarLabel *Labels;
};

Элементы

CountLabels
Количество элементов в массиве Labels.
Labels
Содержит массив структур типа KeyBarLabel, с новыми заголовками функциональных клавиш.
