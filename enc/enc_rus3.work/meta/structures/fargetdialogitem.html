FarGetDialogItem

главная | структуры | Dialog API
Структура FarGetDialogItem используется для запроса информации об элементе диалога в сообщении DM_GETDLGITEM. 
struct FarGetDialogItem
{
  size_t Size;
  struct FarDialogItem *Item;
};

Элементы

Size
Размер поля Item.
Items
Буфер, выделенный плагином для размещения запрашиваемого элемента диалога FarDialogItem.

Замечания

Выделенный буфер должен быть освобожден плагином.
