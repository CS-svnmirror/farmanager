GetContentFieldsInfo

главная | структуры | TODO:Content API
Структура GetContentFieldsInfo передает плагину список типов контент-колонок в функции GetContentFieldsW. 
struct GetContentFieldsInfo
{
  size_t StructSize;
  size_t Count;
  const wchar_t* const *Names;
  void* Instance;
};

Элементы

StructSize
Размер структуры GetContentFieldsInfo.
Count
Количество элементов в массиве Names.
Names
Указывает на массив типов колонок.
