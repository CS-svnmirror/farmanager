ConfigureW

главная | экспортируемые функции
Функция ConfigureW вызывается Far Manager, когда пользователь выбрал в меню "Параметры внешних модулей" пункт, добавленный туда данным плагином. 
int WINAPI ConfigureW(
  const struct ConfigureInfo *Info
);

Параметры

Info
Указатель на структуру ConfigureInfo - информация, переданная Far Manager'ом плагину, о выбранном пункте меню.
В начале функции обязательно проверьте, что поле ConfigureInfo.StructSize >= sizeof(ConfigureInfo).

Возвращаемое значение

Если функция завершается успешно, то возвращается TRUE - в этом случае Far Manager выполняет обновление панелей.
Если пользователь прервал выполнение диалога конфигурации, то верните FALSE.

Замечания

Если ваш плагин содержит только одну функцию (т.е. добавляет один пункт в меню "Параметры внешних модулей"), то аргумент Info можно игнорировать.

Пример

int WINAPI ConfigureW(const struct ConfigureInfo *Info)
{
  return Config();
}
