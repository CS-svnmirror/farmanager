ProcessEditorEventW

главная | экспортируемые функции | Editor API
Функция ProcessEditorEventW сообщает плагину о событиях, происходящих в редакторе Far Manager. 
intptr_t WINAPI ProcessEditorEventW(
  const struct ProcessEditorEventInfo *Info
);

Параметры

Info
Указатель на структуру ProcessEditorEventInfo, Far Manager передает информацию о событии.
В начале функции проверьте, что поле ProcessEditorEventInfo.StructSize >= sizeof(ProcessEditorEventInfo).

Возвращаемое значение

Возвращаемое значение зависит от типа события, смотрите описание событий для уточнения информации.
Для необрабатываемых плагином событий, функция должна возвращать 0.
