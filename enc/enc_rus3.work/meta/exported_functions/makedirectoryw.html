MakeDirectoryW

главная | экспортируемые функции
Функция MakeDirectoryW вызывается Far Manager для создания нового каталога в эмулируемой файловой системе. 
int WINAPI MakeDirectoryW(
  struct MakeDirectoryInfo *Info
);

Параметры

Info
Указатель на структуру MakeDirectoryInfo, плагин передает информацию о создаваемом каталоге.

Возвращаемое значение

В случае успеха возвращаемое значение должно быть равно 1. В случае провала возвращается 0. Если функция была прервана пользователем, то должно возвращаться -1.
