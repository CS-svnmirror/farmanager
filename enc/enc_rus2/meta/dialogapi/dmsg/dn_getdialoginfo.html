DN_GETDIALOGINFO

События | Dialog API
Сообщение DN_GETDIALOGINFO посылается в процедуру обработки диалога, когда FAR желает получить информацию о вашем диалоге.

Param1

0

Param2

Указатель на структуру DialogInfo, в которую плагин должен поместить данные.

Return

Верните TRUE, если заполнили структуру, или FALSE, если проигнорировали.

Пример

// Некоторый сгенерированный GUID, например - {66ffe4cf-ee00-4b45-b91a-163f7c57d084}
static const GUID DlgGUID={0x66ffe4cf,0xee00,0x4b45,{0xb9,0x1a,0x16,0x3f,0x7c,0x57,0xd0,0x84}};

// Заполняем...
LONG_PTR WINAPI MyDialogProc(HANDLE hDlg,int Msg,int Param1,LONG_PTR Param2)
{
  switch(Msg)
  {
    case DN_GETDIALOGINFO:
      if (((DialogInfo*)(Param2))->StructSize != sizeof(DialogInfo))
        return FALSE;
      ((DialogInfo*)(Param2))->Id=DlgGUID;
      return TRUE;
  }
  return Info.DefDlgProc(hDlg,Msg,Param1,Param2);
}
