GetOpenPluginInfoW

главная | экспортируемые функции
Функция GetOpenPluginInfoW вызывается для получения информации об открытом плагине. 
void WINAPI GetOpenPluginInfoW(
  HANDLE hPlugin,
  struct OpenPluginInfo *Info
);

Параметры

hPlugin
Описатель плагина, возвращается функциями OpenPluginW или OpenFilePluginW.
Info
Указатель на структуру типа OpenPluginInfo, которая должна быть заполнена этой функцией.

Возвращаемое значение

Нет.

Замечания

  1. Структура OpenPluginInfo, передаваемая функции GetOpenPluginInfoW, уже заполнена 0. Плагин обязательно должен заполнить поле OpenPluginInfo.StructSize
  2. FAR вызывает функцию GetOpenPluginInfoW достаточно часто, поэтому при проектировании плагина необходимо минимизировать "расчеты", производимые в этой функции.
