SetFindListW

главная | экспортируемые функции
Функция SetFindListW предназначена для переноса найденных объектов в диалоге Поиска Файлов в эмулируемую файловую систему. Физически файлы не должны копироваться или изменяться.
int WINAPI SetFindListW(
  HANDLE hPlugin,
  const struct PluginPanelItem *PanelItem,
  int ItemsNumber
);

Параметры

hPlugin
Описатель плагина, возвращается функциями OpenPluginW или OpenFilePluginW.
PanelItem
Указатель на массив структур PluginPanelItem. Каждая структура описывает файл для помещения в панель.
ItemsNumber
Число элементов в массиве PanelItem.

Возвращаемое значение

В случае удачного завершения возвращаемое значение должно быть TRUE. Иначе функция должна возвращать FALSE.

Замечания

  • Типичный представитель, использующий данную функцию - плагин Temporary panel (TMPCLASS.CPP, функция TmpPanel::SetFindList)
  • Перед вызовом этой функции FAR Manager вызывает функцию OpenPluginW(OPEN_FINDLIST,0) и только после успешного ее завершения вызовет SetFindListW.
