ConfigureW

главная | экспортируемые функции
Функция ConfigureW вызывается, когда пользователь выбрал в меню "Plugins configuration" ("Параметры внешних модулей") пункт, добавленный туда данным плагином.
int WINAPI ConfigureW(
  int ItemNumber
);

Параметры

ItemNumber
Номер выбранного элемента в списке пунктов, экспортированном данным плагином в меню "Параметры внешних модулей".

Возвращаемое значение

Если функция завершается успешно, то возвращается TRUE - в этом случае FAR выполняет обновление панелей.
Если пользователь прервал выполнение диалога конфигурации, то верните FALSE.

Замечания

Если ваш плагин содержит только одну функцию, то параметр ItemNumber можно игнорировать.

Пример

int WINAPI _export ConfigureW(int ItemNumber)
{
  switch(ItemNumber)
  {
    case 0:
      return(Config());
  }
  return(FALSE);
}
