WindowInfo

главная | структуры
Структура WindowInfo содержит информацию о конкретном окне FAR Manager. Плагин может получить информацию, используя команду ACTL_GETWINDOWINFO. 
struct WindowInfo{
  int  Pos;
  int  Type;
  int  Modified;
  int  Current;
  wchar_t *TypeName;
  int  TypeNameSize;
  wchar_t *Name;
  int  NameSize;
};

Элементы

Pos
Номер окна, о котором нужно узнать информацию. Нумерация идет с 0. Pos = -1 вернет информацию о текущем окне.
Type
Тип окна. Может иметь одно из следующих значений (перечисление WINDOWINFO_TYPE):
ТипОписание
WTYPE_PANELS Файловые панели.
WTYPE_VIEWER Окно встроенной программы просмотра.
WTYPE_EDITOR Окно встроенного редактора.
WTYPE_DIALOG Диалог.
WTYPE_VMENU Меню.
WTYPE_HELP Окно подсказки.
Modified
Признак модификации содержимого файла в окне. Может быть только у WTYPE_EDITOR.
Current
Является ли окно текущим.
TypeName
Имя типа окна (Panels,View,Edit,Help,VMenu), зависит от текущего языка.
TypeNameSize
Размер буфера TypeName с учетом завершающего '\0'.
Name
Имя окна. Для окон типа WTYPE_VIEWER, WTYPE_EDITOR содержит имя файла. Для панелей - имя текущего файлового объекта. Для окна помощи - полный путь к просматриваемому HLF-файлу. Для диалогов и меню - заголовок.
NameSize
Размер буфера Name с учетом завершающего '\0'.

Пример

void GetFarWindowInfo()
{
  WindowInfo WInfo;
  int CountWindow;
  int I;
  FILE *Fp;

  if((Fp=_wfopen(L"window.log",L"a+t")) == NULL)
      return ;

  // запросили число окон
  CountWindow=(int)Info.AdvControl(Info.ModuleNumber,ACTL_GETWINDOWCOUNT,NULL);
  fwprintf(Fp,L"WindowCount=%i\n",CountWindow);

  for ( I=0; I < CountWindow; I++ )
  {
    // установили нужный номер
    WInfo.Pos=I;

    WInfo.TypeName=NULL;
    WInfo.Name=NULL;

    // запросили информацию об размере буферов
    Info.AdvControl(Info.ModuleNumber,ACTL_GETWINDOWINFO,(void*)&WInfo);

    WInfo.TypeName=(wchar_t *) malloc(WInfo.TypeNameSize);
    WInfo.Name=(wchar_t *) malloc(WInfo.NameSize);

    // запросили информацию
    Info.AdvControl(Info.ModuleNumber,ACTL_GETWINDOWINFO,(void*)&WInfo);
    
    // распечатали ее
    fwprintf(Fp,L"Window[%i], Type=%i (%s), File=[%s] Current=%i, Modified=%i\n",
               I,WInfo.Type,WInfo.TypeName,WInfo.Name,
               WInfo.Current,WInfo.Modified);
    
    free(WInfo.TypeName);
    free(WInfo.Name);

 }
  fclose(Fp);

  // установили окно номер 1 (то есть второе)
  Info.AdvControl(Info.ModuleNumber,ACTL_SETCURRENTWINDOW,(void*)1);
}
