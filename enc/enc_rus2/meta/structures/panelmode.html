PanelMode

главная | структуры
Структура PanelMode описывает один режим панели (view mode). Массив таких структур может быть передан FAR Manager`у в функции GetOpenPluginInfoW. 
struct PanelMode
{
  const wchar_t *ColumnTypes;
  const wchar_t *ColumnWidths;
  const wchar_t * const *ColumnTitles;
  int FullScreen;
  int DetailedStatus;
  int AlignExtensions;
  int CaseConversion;
  const wchar_t *StatusColumnTypes;
  const wchar_t *StatusColumnWidths;
  DWORD Reserved[2];
};

Элементы

ColumnTypes
Текстовая строка, описывающая типы колонок. Кодируются с помощью одной или нескольких букв, разделённых запятыми, например, так L"N,SC,D,T". Если вы используете стандартные режимы, настроенные в FAR`е, то установите ColumnTypes в NULL.
ColumnWidths

Текстовая строка, описывающая ширину колонок.

ColumnTitles
Указатель на массив строк, описывающих заголовки (название) колонок. Если вы будете использовать стандартные названия для колонок, то установите это значение в NULL.
FullScreen
TRUE - показывать панель во весь экран вместо половины экрана.
DetailedStatus
TRUE - в статусной строке будут отображены имя, размер, дата и время файла, для случая StatusColumnTypes=NULL и StatusColumnWidths=NULL. В противном случае в строке статуса будет отображаться только имя файла.
AlignExtensions
TRUE - выравнивать расширения файлов.
CaseConversion
FALSE - для отмены преобразования файлов имён.
StatusColumnTypes
Подобно ColumnTypes, но применительно для строки статуса. Если вы хотите использовать стандартную строку статуса, установите StatusColumnTypes в NULL.
StatusColumnWidths
Подобно ColumnWidths, но применительно для строки статуса.
Reserved
Зарезервировано для будущего использования. Установите в 0.
