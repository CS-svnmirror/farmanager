ViewerMode

главная | структуры
Структура ViewerMode используется для получения дополнительной информации о текущем состоянии программы просмотра FAR Manager. 
struct ViewerMode
{
   UINT CodePage;
   int Wrap;
   int WordWrap;
   int Hex;
   DWORD Reserved[4];
};

Элементы

CodePage
Кодовая страница символов.
Wrap
Включена (1)/выключена (0) свёртка.
WordWrap
Тип свёртки: 1 - используется перенос строк по словам, 0 - по буквам.
Hex
Если 1, то программа просмотра в hex-режиме.
Reserved
Зарезервировано для будущего использования.
