_FAR_NO_NAMELESS_UNIONS

главная | типы и определения
Макроопределение _FAR_NO_NAMELESS_UNIONS определяет, будут ли в структуре FarDialogItem использоваться неименованные объединения (union). Такие конструкции допустимы стандартом C++, но не в ANSI C.

Если не определено макроопределение _FAR_NO_NAMELESS_UNIONS, структура FarDialogItem будет иметь следующий вид: 

struct FarDialogItem
{
  ...
  union {
    int Selected;
    const wchar_t *History;
    const wchar_t *Mask;
    struct FarList *ListItems;
    CHAR_INFO *VBuf;
  };

};
Таким образом, для обращения, например, к полю Selected структуры FarDialogItem достаточно будет писать - Selected.

Если определено макроопределение _FAR_NO_NAMELESS_UNIONS, в структуре будут использоваться именованные объединения. При этом структура будет совместима с компиляторами ANSI C. Структура будет иметь следующий вид: 

struct FarDialogItem
{
  ...
  union {
    int Selected;
    const wchar_t *History;
    const wchar_t *Mask;
    struct FarList *ListItems;
    CHAR_INFO *VBuf;
  } Param;
  ...
};

В этом случае для обращения, например, к полю Selected структуры необходимо будет писать - Param.Selected.

То или иное макроопределение должна быть объявлено перед директивой #include "plugin.hpp": 

#define _FAR_NO_NAMELESS_UNIONS
#include "plugin.hpp"

Внимание!

  • Умолчанием является вариант, при котором _FAR_NO_NAMELESS_UNIONS не определена.
