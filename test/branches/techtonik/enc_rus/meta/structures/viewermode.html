ViewerMode

главная | структуры
Структура ViewerMode используется для получения дополнительной информации о текущем состоянии программы просмотра FAR. 
struct ViewerMode
{
   int UseDecodeTable;
   int TableNum;
   int AnsiMode;
   int Unicode;
   int Wrap;
   int WordWrap;
   int Hex;
   DWORD Reserved[4];
};

Элементы

UseDecodeTable
Если 1, то используется таблица перекодировки.
TableNum
Номер пользовательской таблицы перекодировки. Имеет смысл, если UseDecodeTable равно TRUE.
AnsiMode
Если 1, то текущая кодировка в программе просмотра - ANSI.
Unicode
Если 1, то текущая кодировка в программе просмотра - Unicode.
Wrap
Включена (1)/выключена (0) свертка.
WordWrap
Тип свертки: 1 - используется перенос строк по словам, 0 - по буквам.
Hex
Если 1, то программа просмотра в hex-режиме.
Reserved
Зарезервировано для будущего использования.
