OpenDlgPluginData

главная | Dialog API
Указатель на структуру OpenDlgPluginData передается в функцию OpenPlugin плагина при вызове его из диалога. 
struct OpenDlgPluginData
{
  int ItemNumber;
  HANDLE hDlg;
};

Элементы

ItemNumber
Номер выбранного пункта в меню из зарегистрированных плагином пунктов.
hDlg
Указатель на диалог, из которого вызван плагин.
